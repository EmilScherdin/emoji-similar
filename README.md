# emoji-similar

Get array of similar Emojis




How to use
----------
```
getSimilar(emoji [, count]);
```
   
Example
```

getSimilar('ðŸ˜„', 4);
// return list of 4 similar emoji

// return list of 3 similar emoji

// return list of all similar emoji

```

Installation
------------
```
npm i -S emoji-similar
```
